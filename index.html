Magic Souls

Magic Souls is a third-person action game where you control a wizard that must restore peace to his valley. It was developed in Unity as the final project of my Computer Graphics course. After the course was over, my team and I decided to polish it a little and publish it on itch.io. I was in charge of the shop system, UI, and enemy AI. I also composed the music of the game using FL Studio.

Click here to see the project!

Click here to download the game!

VR Serious Video Game

This is my dissertation project where I designed and developed a virtual reality serious video game for teaching structural engineering concepts. The game was developed in Unreal Engine 5 and tested by a group of 25 civil engineering students from the university. The results prove that VR serious video games can be effectively used for teaching or reinforcing structural engineering concepts.

Click here to see the project!

Pong Proximal Policy Optimization Self-Play Agent using Unity ML-Agents

This project is a comparison of using PPO vs using PPO with competitive self-play for creating an agent that plays Pong using Unity ML-Agents. This was developed as my final project for my intelligent systems course.

Click here to see the project!

GPU Flocking

This project is a comparison of flocking AI that runs on the CPU vs one that runs on the GPU. This was developed using Unity as the final project of my Programming Languages course.

Click here to see the project!

Origins

Origins is an action game where you control a native Aztec during the first Viceroyalty of New Spain around the year 1600. You must survive as many timed rounds as you can killing any enemy colonizer that crosses your path. It was developed using Java as my final team project for my Video game project course. In charge of the enemy AI.
